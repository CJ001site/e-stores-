# 🏪 Multi-Mode Marketplace

A dynamic, mobile-responsive e-commerce single-page application styled with a clean, user-friendly interface reminiscent of top marketplace platforms. This application seamlessly bridges the gap between everyday shoppers and independent creators.

## 🚀 Key Features

*   **Dual-Account Modes**: Toggle easily between **User Mode** (for quick browsing and shopping) and **Creator Mode** (for verified sellers).
*   **Secure Seller Verification**: Creator features are protected behind a private access code (`12345`).
*   **Dynamic Custom Storefronts**: Once a creator registers their business name, the site header instantly transforms into a stylish, custom handwritten logo for their brand.
*   **Cloud-Synced Inventory**: Integrated with a live **Google Sheets Database API** to fetch and permanently save product posts in real time without a complex backend server.
*   **Instant Communication**: Equipped with an orange emergency/order banner that dynamically updates to display the active seller's phone number, alongside native one-tap email ordering.

## 🛠️ Tech Stack

*   **Frontend**: HTML5, CSS3 (Flexbox/Grid), JavaScript (ES6)
*   **Database/Backend**: Google Apps Script & Google Sheets API
*   **Fonts**: Google Fonts (Poppins & Caveat)
